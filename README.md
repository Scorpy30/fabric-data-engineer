# Fabric Data Engineer Learning Repo

This repository contains my study notes, exercises, and resources as I prepare for the **Fabric Data Engineer Associate (DL-700)** certification and follow along with **Data Days** sessions.  
I maintain two parallel tracks:
- **Certification** → Official DL-700 modules from Microsoft Learn  
- **Data Days** → Community-driven sessions, organized here by module/topic names  

---

## 📂 Structure

- `certification/` → Notes aligned with DL-700 modules  
- `data-days/` → Notes aligned with Data Days sessions (organized by module names, not days)  
- `crosswalk/` → Mapping between certification modules and Data Days coverage  
- `projects/` → Hands-on labs, demos, and experiments  
- `resources/` → Cheat sheets, references, and external links
